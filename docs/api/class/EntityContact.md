## 属性
[other](property): [](Box3Entity) / [](GameEntity)
:   碰到的实体

[axis](property): [](Box3Vector3) / [](GameVector3)
:   碰撞的分离轴，也就是碰撞后物体弹飞的 **方向**

[force](property): [](Box3Vector3) / [](GameVector3)
:   碰撞力大小
